# Arjuna App

Aplikasi ini terdiri dari dua bagian utama:

## 📁 Struktur Proyek

```
arjuna-app/
├── backend/    # REST API & server-side logic
└── frontend/   # Antarmuka pengguna (web)
```

## 🚀 Backend

Berisi kode server-side, REST API, dan logika bisnis aplikasi.

## 🖥️ Frontend

Berisi kode antarmuka pengguna berbasis web.

## ⚙️ Cara Menjalankan

### Backend
```bash
cd backend
# ikuti instruksi di dalam folder backend
```

### Frontend
```bash
cd frontend
# ikuti instruksi di dalam folder frontend
```
